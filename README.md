# Конструктор писем

### Версии
```
node -v = v14.17.0
yarn -v = 1.22.10
npm -v = 7.16.0
```

### Установка пакетов
```
yarn install
```

### Запуск в режиме разработки
```
yarn serve
```

### Сборка проекта
```
yarn build
```

### Запуск юнит тестов
```
yarn test:unit
```

### Для работы редактора tiny необходимо в корне проекта создать файл .env 
```
VUE_APP_TINY_API_KEY=тут_апи_ключ_tiny
```
